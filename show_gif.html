Display a gif

Usage

show_gif(filename, method = "browser")

Arguments

filename
Filename of gif
method
One of "browser", "shiny" or "gadget". If you have a recent version of shiny and are running in Rstudio, use method "gadget" to open the gif in the viewer window.

Description

Display a gif using shiny
